The `fast_float.h` C++ header provides functions for quickly parsing floating point numbers from strings.

This library is licensed under the Apache 2.0 license (in addition to MIT and others, see header file).

The copy included here is version 8.1.0 and was downloaded from https://github.com/fastfloat/fast_float/releases/download/v8.1.0/fast_float.h.

This library was originally added for use in processing VTK based inflow files in the AWAE module.
